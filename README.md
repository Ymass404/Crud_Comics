# Crud_Comics

Crud_Comics est une application web CRUD (Create, Read, Update, Delete) permettant de gérer une collection de comics.

Ce projet a pour objectif de mettre en place une architecture moderne et propre autour de Next.js, TypeScript et Drizzle ORM, tout en restant simple et pédagogique.

---

## Présentation

L’application permet de :

- Créer un comic
- Consulter la liste des comics
- Modifier un comic existant
- Supprimer un comic

Elle constitue une base de travail pour comprendre la mise en place d’un projet full stack structuré avec une base de données et une organisation claire du code.

---

## Stack technique

- **Next.js**
- **TypeScript**
- **Drizzle ORM**
- **PostgreSQL**
- **Docker / Docker Compose**
- **CSS**

---

## Architecture

Le projet est organisé de manière à séparer :

- La configuration (Next.js, Drizzle)
- La gestion de la base de données
- La logique applicative
- L’interface utilisateur

Cette structure permet de garder un code maintenable, évolutif et clair.

---

## Objectif pédagogique

Crud_Comics a été conçu pour :

- Comprendre le fonctionnement d’un CRUD complet
- Manipuler une base de données avec un ORM moderne
- Travailler avec TypeScript dans un environnement réel
- Structurer un projet web proprement

Ce dépôt peut servir de base pour évoluer vers :

- Authentification
- Pagination
- Filtres de recherche
- Tests unitaires et end-to-end
- CI/CD
- Déploiement cloud

---

Projet développé dans un objectif d’apprentissage et de montée en compétence sur l’écosystème Next.js et les architectures modernes.
